# Agente de Treinamento

Gera roleplays, quizzes e simulações para treinar corretores.

## Responsabilidade
Criar exercícios práticos (roleplay de objeção, quiz de crédito) sob demanda.

## Entradas
- Tema (crédito, objeção, documentação), nível do corretor.

## Saídas
- Roleplay estruturado (cliente x corretor).
- Quiz com gabarito.

## Base de conhecimento
- [[Biblioteca de Roleplays]], [[Objeções e Respostas]], [[Treinamento de Crédito Caixa]].

## Erros comuns
- Cenários irreais para o público MCMV.

## Melhorias futuras
- Correção automática de quizzes.
