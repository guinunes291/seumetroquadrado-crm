# Padrões de Prompts

Como escrever prompts consistentes para os agentes da SMQ. Base para novos agentes (ver [[Template - Prompt]]).

## Estrutura padrão de um prompt
1. **Papel** — quem o agente é (ex.: "assistente de qualificação da SMQ").
2. **Objetivo** — o que ele precisa alcançar.
3. **Contexto** — informação da SMQ e do lead.
4. **Regras** — o que sempre/nunca fazer.
5. **Formato de saída** — como responder.

## Regras de ouro da SMQ (todos os agentes)
- Nunca prometer aprovação de crédito ou subsídio.
- Sempre tratar valores como estimativa.
- Tom acolhedor, objetivo e sem juridiquês.
- Confirmar dados sensíveis antes de avançar.
- Encaminhar para humano quando fugir do escopo.

## Boas práticas
- Um agente, um objetivo.
- Exemplos (few-shot) quando o comportamento for específico.
- Versione mudanças relevantes de prompt no [[Roadmap Squad IA]].
