# SMQ Operating System

Vault Obsidian da **Seu Metro Quadrado (SMQ)** — o sistema central de organização, memória operacional e base de conhecimento da empresa.

Imobiliária focada em **Minha Casa Minha Vida**, primeiro imóvel, lançamentos, qualificação de clientes, análise de crédito, documentação, follow-up, CRM, Meta Ads, WhatsApp, treinamento de corretores e uso intenso de IA.

---

## Para que serve este vault

- Documentar o **CRM** (páginas, bugs, melhorias, decisões, roadmap).
- Organizar o **Squad de IA** (agentes, prompts, integrações, erros comuns).
- Manter a **base de conhecimento comercial** (scripts, objeções, crédito).
- Estruturar **treinamentos**, **processos (SOPs)**, **reuniões** e **decisões**.
- Preparar tudo para conectar futuramente com **Claude, Cowork, n8n, MCP e CRM**.

---

## Como usar (versão gratuita do Obsidian)

Este vault funciona 100% local, em Markdown puro. **Não depende de Obsidian Sync, Publish ou plugins pagos.**

1. Instale o Obsidian gratuito: https://obsidian.md
2. Copie a pasta `SMQ Operating System` para o seu computador, ex.: `~/Documents/Obsidian/SMQ Operating System`.
3. No Obsidian: **Open folder as vault** → selecione a pasta `SMQ Operating System`.
4. Abra `00 - Dashboard/Home.md` e comece a navegar pelos links `[[...]]`.

> Este vault foi versionado dentro do repositório do CRM na branch de setup do Obsidian. Você pode mantê-lo no Git (backup grátis) ou apenas copiar a pasta para o Obsidian local.
