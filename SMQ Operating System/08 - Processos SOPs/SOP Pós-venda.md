# SOP Pós-venda

**Objetivo:** acompanhar o cliente após a assinatura, garantir satisfação e gerar indicações.

## Passo a passo
1. Confirmar assinatura e próximos passos com o cliente.
2. Acompanhar prazos (obra/entrega, se aplicável).
3. Fazer check-ins programados.
4. Pedir avaliação/depoimento (marketing — [[Aprendizados de Campanhas]]).
5. Solicitar indicações.

## Regras
- Cliente satisfeito = fonte de novas vendas.
- Registrar contatos de pós-venda no CRM.
