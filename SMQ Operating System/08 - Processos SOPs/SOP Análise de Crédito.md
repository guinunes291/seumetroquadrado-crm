# SOP Análise de Crédito

**Objetivo:** conduzir a análise de crédito do cliente de forma correta até a resposta da Caixa.

## Passo a passo
1. Coletar dados: renda, FGTS, tempo de carteira, situação cadastral.
2. Fazer simulação inicial (estimativa — ver [[Treinamento de Crédito Caixa]]).
3. Montar a pasta ([[SOP Documentação]]).
4. Submeter à Caixa.
5. Acompanhar retorno e comunicar o cliente.
6. Se aprovado → seguir para [[Fechamento SMQ]].
7. Se pendente → resolver documentação/condição.

## Regras
- Tudo é estimativa até a resposta oficial.
- Comunicar o cliente com transparência sobre prazos.
