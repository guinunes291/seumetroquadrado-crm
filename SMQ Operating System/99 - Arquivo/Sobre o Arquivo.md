# Sobre o Arquivo

Esta pasta guarda documentos **arquivados** — que não estão mais em uso, mas têm valor histórico.

## Regras
- Ao aposentar um documento, mude o `status:` para `arquivado` e mova-o para cá.
- Não apague — arquive. Histórico é memória.
- Mantenha os links; use `[[...]]` normalmente.
