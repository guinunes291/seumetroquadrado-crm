# Página de Leads

Onde os leads entram, são qualificados e distribuídos.

## O que essa página faz
- Recebe leads de Meta Ads / WhatsApp.
- Mostra dados: nome, telefone, origem, renda, objetivo.
- Permite qualificar (ver [[Qualificação de Cliente]]).
- Distribui para corretores (ver [[SOP Distribuição de Leads]]).

## Campos importantes
- Origem da campanha
- Renda / composição de renda
- FGTS / tempo de carteira
- Objetivo (morar, investir, sair do aluguel)
- Status no funil

## Dores conhecidas (mover confirmadas para [[Bugs Conhecidos CRM]])
- (a preencher) Muitos cliques para qualificar um lead.
- (a preencher) Falta de filtro rápido por origem de campanha.
