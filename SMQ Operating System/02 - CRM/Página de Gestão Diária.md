# Página de Gestão Diária

Onde o corretor organiza o dia: tarefas, follow-ups e prioridades.

## O que essa página faz
- Mostra follow-ups do dia.
- Lista leads que precisam de ação.
- Ajuda a priorizar quem está quente.

## Rotina esperada (ver [[Rotina Comercial Diária]])
1. Abrir a gestão diária no início do dia.
2. Resolver follow-ups atrasados primeiro.
3. Atacar leads qualificados quentes.
4. Registrar cada contato.

## Melhorias sugeridas
- (a preencher) Reduzir cliques para marcar follow-up concluído → [[Backlog de Melhorias CRM]].
