# Página de Projetos

Cadastro e gestão dos empreendimentos/imóveis ofertados pela SMQ.

## O que essa página faz
- Lista projetos/lançamentos disponíveis.
- Guarda dados: localização, faixa de preço, subsídio, tipologia.
- Serve de base para a [[Ofertas por Projeto]] do marketing.

## Campos importantes
- Nome do projeto / construtora
- Faixa de renda atendida (MCMV)
- Valor a partir de / subsídio estimado
- Documentação exigida
- Status (lançamento, em obras, pronto)

## Uso pelo comercial
- Corretor consulta para ofertar imóvel compatível com o perfil.
- Base para argumentação e [[Fechamento SMQ]].
